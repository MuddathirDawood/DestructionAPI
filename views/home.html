Destruction Weapons of Annihilation

ALL ENDPOINTS

Method Route Description
GET /users Gets All Users in the database
GET /users/:id Gets a SINGLE User in the database
POST /users Adds a User into the database also called the Register
PATCH /users It reads the Users and finds the one with the details entered also called the Login
DELETE /users/:id Deletes a single user from the database
PUT /users/:id It edits anything pertaining to a single User
GET /weapons Gets All weapons in the database
GET /weapons/:id Gets a SINGLE User in the database
POST /weapons Adds a weapon into the database
GET /weapons/era/:eraID Gets all weapons that corresponds with their respective eraID
PUT /weapons/:id It edits anything pertaining to a single Weapon
DELETE /weapons/:id Deletes a single weapon from the database
GET /eras Gets All eras in the database
GET /eras/:id Gets a SINGLE era in the database
GET /eras/weapons/:id Gets 3 weapons from the selected era
PUT /eras/:id It edits anything pertaining to a single era
